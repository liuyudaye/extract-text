说明：

此页面展示了如何使用鸿蒙字体（HarmonyOS Sans SC Bold）来显示"引领AI出版新范式"这几个字。

通过 CSS 的 @font-face 规则加载字体文件，并使用 .harmony-text 类来应用字体。

字体文件使用 Fontmin 提取的子集版本，只包含目标文字，文件大小从 8MB 减小到约 10KB（WOFF2）。

字体文件位于 public/font-subset/ 目录。

提取字体子集：运行 npm run extract-font 可以重新提取字体子集。
